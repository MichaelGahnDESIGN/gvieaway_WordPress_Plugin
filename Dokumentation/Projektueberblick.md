# Projektüberblick

MGD Giveaway ist ein WordPress Plugin zum Erstellen von Formularen für Gratis-eBooks und PDF-Downloads. Nach erfolgreicher Anmeldung wird ein Download-Button angezeigt. Optional kann der Download-Link per E-Mail versendet werden. Neue Anmeldungen werden in einer Mail-Liste gespeichert und an eine einstellbare Empfängeradresse gemeldet.
