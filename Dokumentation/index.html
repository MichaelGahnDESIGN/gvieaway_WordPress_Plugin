MGD Giveaway

WordPress Plugin fuer Giveaway-Formulare mit Download-Link nach Anmeldung.

Ueberblick Funktionen Setup Risiken Kosten & Lizenzen Versions-Log

Ueberblick

Das Plugin erstellt Formulare, speichert Anmeldungen lokal und zeigt nach erfolgreicher Anmeldung einen Download-Button fuer eine Datei aus der WordPress-Mediathek.

Funktionen

  • Dashboard mit Formularliste
  • Formular-Editor mit Feldtypen
  • Shortcode [mgd_giveaway id="123"]
  • SMTP oder PHP-Mail
  • Credits-Seite

Setup

ZIP installieren, Plugin aktivieren, Formular erstellen, Datei auswaehlen und Shortcode in eine Seite einfuegen.

Risiken

Personenbezogene Daten, E-Mail-Versand, Speicherfristen und Einwilligungen muessen vor produktivem Einsatz rechtlich geprueft werden.
